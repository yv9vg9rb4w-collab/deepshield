// 03 — Architecture
Stack
Technique
🤖
Claude Vision
Moteur IA

Modèle multimodal d'Anthropic. Analyse visuelle avancée, compréhension du contexte et explication des anomalies.

🌐
Anthropic API
Backend IA

Appels directs à l'API /v1/messages depuis le navigateur. Pas de serveur intermédiaire. Résultats en temps réel.

📄
HTML / CSS / JS pur
Frontend

Un seul fichier HTML sans dépendances. Drag & drop natif, FileReader API, fetch vers l'API Anthropic.

Encodage local en base64 avant envoi. Supporte JPG, PNG, WEBP, GIF jusqu'à 5 Mo.

La clé API est chiffrée et stockée localement dans le navigateur. Jamais transmise à un tiers. Persistante entre sessions.

Hébergement gratuit. Un seul fichier index.html — pas de build, pas de serveur. Déploiement en 2 minutes.
