// 02 — Fonctionnalités
Ce que
fait DeepShield
01
🧠
Réseau Neuronal

EfficientNet-B4 fine-tuné sur FaceForensics++ et DFDC. Classification binaire réelle/fausse avec score de confiance.

02
📊
Analyse Fréquentielle

DCT et FFT pour détecter les artefacts spectraux invisibles laissés par les réseaux génératifs GAN lors de la synthèse.

03
🎭
Landmarks 468 pts

MediaPipe extrait 468 points faciaux. Analyse géométrique et temporelle pour détecter les incohérences de déformation.

04
🎬
Vidéo Frame-by-Frame

Extraction intelligente des frames, analyse par batch et graphe temporel du score de détection pour toute la vidéo.

05
🔗
Analyse par URL

Collez directement un lien image, vidéo ou page web. DeepShield extrait et analyse automatiquement le contenu visuel sans téléchargement.

06
🗺
Heatmap Zones Suspectes

Grad-CAM visualise les zones du visage ayant déclenché la détection. Export rapport PDF complet avec toutes les métriques.

07
🌐
Scan de Pages Web

Analyse de l'intégralité d'une page : toutes les images et vidéos embarquées sont scannées et un rapport global est généré.

08
🔒
100% Privé

Aucun fichier stocké sur nos serveurs. Traitement sécurisé côté serveur avec suppression automatique après analyse.

// 03 — Architecture
Stack
Technique
🐍
Python + FastAPI
Backend & API

API REST ultra-rapide avec FastAPI. Gestion des uploads, scraping d'URLs, inférence async, retour des résultats en JSON.

🔥
PyTorch + EfficientNet
Deep Learning

EfficientNet-B4 fine-tuné sur FF++. Data augmentation avancée. Export ONNX pour inférence optimisée.

⚛️
React + Tailwind
Frontend

Interface réactive avec drag & drop, saisie URL, visualisation temps réel, Recharts pour les graphes de résultats.

👁
OpenCV + MediaPipe
Vision

Extraction de frames vidéo, détection faciale robuste, 468 landmarks pour l'analyse géométrique.

🕷
Playwright + BeautifulSoup
Scraping URL

Extraction automatique du contenu visuel depuis n'importe quelle URL : images, vidéos, iframes. Rendu JS complet pour les SPAs.

🐋
Docker + CI/CD
Déploiement

Conteneurisation complète. GitHub Actions pour CI/CD automatisé. Déploiement sur Render ou AWS.

📦
FaceForensics++ / DFDC
Dataset

Entraînement sur FF++ (1000+ vidéos) et DeepFake Detection Challenge de Meta. 50k+ images synthétiques.
